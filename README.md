# 🎬 담소 Frontend

Next.js 기반으로 개발된 담소(살아있는 회고록) 프론트엔드 프로젝트입니다.
부모와 자녀가 서로에게 묻고 영상으로 답하며, AI가 그 대화를 인생네컷 기록으로 정리해주는 웹앱입니다.

## 📦 Tech Stack

- Next.js 16
- TypeScript 5
- TailwindCSS
- PWA (next-pwa)
- Axios
- Zustand – 상태 관리
- ESLint – 코드 품질 관리

## 🛠️ 프로젝트 실행

### 1. 요구 사항

- Node.js 22.x (`nvm use` 로 자동 설정)

### 2. 패키지 설치

```bash
npm install
```

### 3. 환경변수 설정

```bash
cp .env.example .env
```

### 4. 개발 서버 실행

```bash
npm run dev
```

### 5. 빌드

```bash
npm run build
npm run start
```
